Panpipe

Version 0.1
A native Pandora Internet Radio client for Ubuntu Touch
Panpipe is an open source GPLv3 client for Panpipe Internet Radio. It is written in QML/JavaScript and is designed to run natively on Ubuntu Touch. In the future I hope to bring Panpipe to other platforms that support QML/JS and are in need of a Pandora client. Panpipe is still under heavy development and has a ways to go to feature completion.
  • Free and Open Source
  • Interactive and intuitive UI

About

Panpipe is one of the growing number of applications designed to integrate natively with the Ubuntu Touch design approach. The focus is on content and controls are tucked away in a toolbar that hides along the bottom edge. Panpipe strives to be intuitive, easy to use, and pleasing to the eye.

Panpipe is open source, and code is hosted on Github. Bug reports and code contributions are all very welcome.

Panpipe is not affiliated with or endorsed by Pandora Media, Inc.
